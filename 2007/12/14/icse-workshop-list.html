Dec 14, 2007 - ICSE Workshop List

The list of workshops for next year's International Conference on Software Engineering (ICSE'08) has been announced. The titles alone provide a lot of insight into what academic software engineering is interested in these days. They also show just how misleading it is to talk about "academic software engineering" as if it was one community, with one set of interests...
  • Socio-Technical Congruence
  • Advances and Applications of Problem Frames
  • Scrutinizing Agile Practices
  • Software Architectures and mobility
  • Software Quality
  • Mining Software Repositories
  • Modeling in Software Engineering
  • Ultra-Large-Scale Software-Intensive Systems
  • Leadership and Management in Software Engineering
  • The Role of Abstraction in Software Engineering
  • Systems Development in SOA Environments
  • Automation of Software Test (no web site yet)
  • Multicore Software Engineering
  • Aspect-Oriented Requirements Engineering and Architecture Design
  • Graph and Model Transformation
  • End-User Software Engineering (no web site yet)
  • Software Development Governance
  • Predictor Models in Software Engineering
  • Software Engineering for Adaptive and Self-Managing Systems
  • Sharing and Reusing Architectural Knowledge
  • Human and Cooperative Aspects of Software Engineering
  • Software Engineering for Computational Science and Engineering
  • Business Impact of Process Improvements (no web site yet)
  • Comparison and Versioning of Software Models
  • Software Engineering for Secure Systems
  • Aerospace Software Engineering

How Learning Works

  • Started reading the education literature in 2011
  • We know a lot about how people learn and how best to teach them
  • But most faculty don't even know this knowledge exists

For Example

  • We have known for 20 years that peer instruction outperforms classic lecturing
  • But no major MOOC platform supports peer instruction
  • Absence isn't technical: a GSoC student built a proof of concept in 12 weeks in 2014

Computer Science Strong Opinion

  • Helped edit similar collection in 2010 on empirical studies in software engineering
  • We know a lot about software and how it's built
  • But almost none of this is in the undergraduate curriculum
  • And students aren't even shown the methods

For Example

  • C-style languages are as hard to learn as ones with random syntax
  • Python and Ruby are significantly easier…
  • …and A/B-tested languages are even easier
  • But no major language tests new features on users before deploying them
